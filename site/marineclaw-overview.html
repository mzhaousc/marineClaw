marineClaw

marineClaw is a local AI agent system for marine genomics and sequence-first bioinformatics. It uses a skill-organized architecture and a generate-execute loop to produce runnable analysis scripts.

Core runtime supports Python, R, and Bash, with web UI, script extraction, and workflow-oriented output.

Key Capabilities

  • BLAST-driven sequence search and hit filtering
  • Multiple sequence alignment and phylogenetic analysis
  • Functional/domain/pathway annotation and ortholog/HGT screening
  • Marine metagenomics profiling and comparative genomics
  • Galaxy Bridge integration (tool discovery + planemo execution path)
  • AlphaFold-compatible API structure prediction from FASTA
  • Marine literature and public dataset retrieval workflows
  • Ecological community stats (PCA/NMDS/PERMANOVA-ready)
  • Reproducible workflow templates (Snakemake and Nextflow DSL2)

Architecture Overview

  • marine_agent/: LangGraph-based agent runtime, config, retriever, executor, CLI, web UI
  • skills/: marine analysis skills as markdown + scripts
  • environment.yml: full bioinformatics environment
  • environment-minimal.yml: lightweight UI/agent startup environment

Quick Start

Minimal UI setup:

conda env create -f environment-minimal.yml && conda activate marineclaw-minimal && python -m marine_agent --web

Full environment setup:

conda env create -f environment.yml && conda activate marineclaw

Publishing Notes

  • Do not publish .env or API keys.
  • Use .env.example for public configuration template.
  • Keep large raw sequencing files outside the git repository.
